One-time setup (2 min):
  1. Go to developer.spotify.com/dashboard
  2. Create an app (any name). Under "Which API/SDKs are you planning to use?" select Web API.
  3. In app Settings → Redirect URIs, add exactly:
    http://127.0.0.1:8888/album-rank.html
  4. Copy your Client ID and paste below
  5. Serve this file: open Terminal, cd to the folder with this file, run:
    python3 -m http.server 8888 --bind 127.0.0.1
  6. Open http://127.0.0.1:8888/album-rank.html
